Objetivo
Crie um programa em JavaScript que solicite ao usuário várias informações e, em seguida, manipule e exiba esses dados de diferentes maneiras.

Requisitos
Peça ao usuário para inserir seu nome completo. Peça ao usuário para inserir sua idade. Peça ao usuário para inserir seu endereço de e-mail.

Tarefas
Exibir o número total de caracteres no nome completo. Exibir o nome completo em maiúsculas e minúsculas. Verificar se o e-mail contém "@" e ".". Criar uma saudação personalizada usando os dados fornecidos.
